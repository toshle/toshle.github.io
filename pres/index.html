Visum

Augmented reality mobile and web app

Created by Rosen Rachev

Heads Up

Visum is an application for easily exploring, orientation or just learning cool stuff about geographical objects.

What can you do?

Map

Navigate through an interactive map with points of interest.

Augmented reality

Orient yourself with the augmented reality.

And more...

  • News
  • New POIs
  • Videos
  • Pictures
  • Interact with other users

Technologies

  • Lots of javascript
  • Sencha Touch 2
  • node.js
  • MongoDB
